# yahc-app
Yet Another HTTP Client (but mobile)

```
$ ionic cordova plugin add cordova-plugin-app-version # app version
$ ionic cordova plugin add cordova-plugin-network-information # network info
$ ionic cordova plugin add cordova-sqlite-storage # storage plugin
$ ionic cordova plugin add cordova-plugin-inapppurchase # IAP
$ ionic cordova platform add <platform> # platform options or ios or android
```
